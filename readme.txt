== Description ==
amazing neo for elementor add amazing neo icon widgets to Elementor page builder.


All the widgets are highly customizable, you can have icon + title + description, or you can hide each one of them.
you need just a few seconds from getting the best solution for your website.
all you need to do is click on the install / download button and 
amazing neo for elementor will be in your elementor dashboard ready to use!

In the pro version of the plugin you can get evan more...
for more details on the pro version click the link [click the link](https://amazingneo.com "Amazing Neo For Elementor")

== Installation ==
Install like any other plugin, directly from your plugins page.

1. Upload the plugin files to the `/wp-content/plugins/amazing-neo-for-elementor` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
